# 🐢 Turtle Racing Game

Petit projet Python amusant qui simule une **course de tortues aléatoire** à l’aide de la bibliothèque graphique **Turtle**.
Chaque tortue avance de manière aléatoire jusqu’à ce qu’un gagnant soit déterminé.

---

## 📌 Description

Ce projet est un mini-jeu interactif développé en Python.
L’utilisateur choisit le nombre de tortues participantes, puis une course démarre.
Chaque tortue avance de façon aléatoire jusqu’à ce que l’une d’entre elles atteigne la ligne d’arrivée.

Le gagnant est ensuite affiché dans la console.

---

## 🚀 Fonctionnalités

* Interface graphique simple avec la bibliothèque **Turtle**
* Choix du nombre de participants
* Mouvement aléatoire des tortues
* Affichage du gagnant
* Couleurs générées dynamiquement
* Simulation visuelle d’une course

---

## 🛠️ Technologies utilisées
